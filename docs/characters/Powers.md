# Powers

## Power Traits

### Speed
The amount of actions and/or shifts required to use the power

### Entropy Trigger
What happens when the results set of the power roll contains a 1.

### Threshold
The number of Hits at which the magnitude of the power's effect increases.

### Effect
What the power does.
